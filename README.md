## USaaS Admin 插件式快速开发框架

### 基于Vue3+、Vite、Ant Design,以插件形式开箱即用.

### 技术栈

|      版本       | 技术栈  | 备注 |
|  ----------      | -------------  | -------------  |
| 后端PHP版  | PHP8+、Swoole4.5+、hyperf2.2+、MySql5.7+  | 研发中，正式环境请勿使用 |
| 后端NodeJs版  | node14+typescript4.3.5+nestjs8.0+MySql5.7   | 待发布|
| 前端  | Vue3+、Vite、Ant Design | |
